# Minecraft OBJ Model Viewer

Простая и быстрая утилита на Python для просмотра .obj моделей Minecraft с поддержкой .mtl материалов и текстур.

## Особенности
* **Drag-and-Drop**: Просто перетащите .obj файл в окно.
* **Minecraft Style**: Текстуры отображаются без размытия (Nearest Neighbor filtering).
* **Темы**: Поддержка темной, серой и светлой тем оформления.
* **Быстрый сброс**: Клавиша `R` или `Space` возвращает камеру в центр.
* **Прозрачность**: Корректное отображение прозрачных областей в PNG.

## Как запустить

### Способ 1: Готовый файл (EXE)
1. Перейдите в папку `dist` (или в раздел Releases на GitHub).
2. Запустите `MC_OBJ_Viewer.exe`.
3. Перетащите вашу модель в окно программы.
*Примечание: .obj, .mtl и .png должны лежать в одной папке.*

### Способ 2: Запуск из исходного кода (Python)
